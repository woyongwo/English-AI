프로젝트 개요

문제 정의 · 목표 · 핵심 아이디어

평범한 일상 단어조차 순간적으로 떠오르지 않아 불편함을 느끼는 경험을 해결하기 위한 개인화 학습 도구입니다. 생성형 AI와 간격 반복(SRS), 맥락 기반 회상을 결합하여 사용자가 잊기 쉬운 단어를 추적하고 꾸준히 복습하도록 돕습니다.

핵심 기능

MVP 권장: SRS + 컨텍스트 프롬프트

간격 반복 스케줄러 (Spaced Repetition)

사용자별 회상 성공률을 기록해 다음 복습 시점을 계산합니다. 오프라인 캐시(IndexedDB) 및 서버 동기화 지원.

맥락 기반 생성형 프롬프트

직장 이메일, 채팅, 일기 등 실제 문맥에 맞춘 예문과 빈칸 문제를 AI가 생성하여 회상을 유도합니다.

멀티모달 연계 (이미지·발음·예문)

단어마다 이미지, TTS 발음, 예문을 연계해 다양한 경로로 기억을 강화합니다.

개인화 대시보드 & 통계

학습 히트맵, 회상 성공률, 잊음 곡선 예측을 시각화하여 동기 부여를 제공합니다.

아키텍처(초안)

클라이언트·서버·AI 레이어
  • 클라이언트: React 기반 SPA + IndexedDB(로컬 우선)
  • 서버: Node.js API (사용자 동기화, SRS 계산)
  • AI 레이어: 외부 생성형 모델 연동 + 로컬 캐시
  • 데이터: 암호화된 페이로드로 서버 동기화

다음 단계

우선순위 · 프로토타입 · 지표
  1. MVP 핵심 기능 확정: SRS + 컨텍스트 프롬프트
  2. 프론트: React + IndexedDB, 백엔드: Node.js + Express
  3. 프로토타입 → 소수 사용자 테스트 → 지표 수집
