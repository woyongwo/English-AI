영어로 한 달 살기

상황 · 수준 · 목적에 따라 필요한 영어 문장과 단어를 자동 생성하고, 30일 학습 플랜으로 매일 5문장씩 부담 없이 학습하도록 돕는 웹 서비스입니다.
핵심은 “암기”가 아니라, 실제로 써먹을 수 있게 만드는 문맥 중심 학습입니다.

학습 분량
30일 · 하루 5문장
개인화 입력
목적 · 레벨 · 상황
학습 관리
⭐ 암기 체크 · 복사
기술 포인트
JSON 응답 강제
맞춤형 플랜 실용 표현 초개인화 웹앱

프로젝트 개요

문제 · 계기 · 목표

영어를 자주 쓰지 않으면, 간단한 표현도 순간적으로 떠오르지 않는 불편함이 생깁니다. 기존 학습 앱은 범용 커리큘럼이 많아, 개인의 목적/상황에 맞는 “내 문장”을 만들기 어렵다는 한계가 있었습니다.

그래서 이 프로젝트는 생성형 AI를 활용해 사용자의 입력(목적/레벨/상황)을 바탕으로 바로 써먹을 문장과 단어를 생성하고, 30일 플랜으로 학습 흐름을 잡아주는 것을 목표로 했습니다.

실생활 영어의 어려움

평소 영어를 거의 쓰지 않으면, 알고 있던 표현도 급하게 말하려 할 때 “막히는” 순간이 생깁니다.

AI 가능성 발견

상황과 수준에 맞는 문장을 AI가 만들어줄 때, 학습이 “암기”가 아니라 “사용”으로 바뀌는 경험을 얻었습니다.

기존 앱의 한계

개인 상황·목적 기반 학습이 부족해, 필요한 표현을 그때그때 모으고 정리하는 데 시간이 많이 듭니다.

이 프로젝트가 하는 일

입력값만으로 30일 플랜을 자동 생성하고, 매일 5문장씩 반복/복습 가능한 형태로 제공합니다.

주요 기능

플랜 생성 · 학습 관리

AI 학습 플랜 자동 생성

사용자의 목적/레벨/상황을 반영해 Day1~Day30 형식으로 계획을 생성합니다. 매일 5문장씩 구성해 부담을 줄이고, 꾸준히 이어지도록 설계했습니다.

문장별 ⭐ 암기 완료 체크

문장 단위로 체크할 수 있어 “얼마나 했는지”가 명확합니다. 학습 진행률을 눈에 보이게 만들어 동기부여를 높입니다.

전체 문장 복사 기능

하루 분량을 한 번에 복사해 메모앱/노션/카톡으로 옮길 수 있습니다. 실제 사용 흐름을 빠르게 만드는 기능입니다.

단어 학습 모드 전환

문장 학습뿐 아니라, 핵심 단어만 따로 보고 싶을 때 모드를 전환할 수 있게 구성했습니다.

입력값 & 생성 흐름

개인화의 핵심
1
학습 목적 선택 여행 · 비즈니스 · 일상 등 원하는 학습 목표를 정합니다.
2
영어 수준 선택 초급/중급/고급 중 자신의 레벨에 맞춰 문장 난이도를 조절합니다.
3
개인정보 입력 나이, 시작 날짜 등을 입력해 ‘학습 시작’ 흐름을 맞춥니다.
4
상황 입력 영어를 실제로 사용할 상황을 자유롭게 적으면, AI가 그 문맥에 맞춰 문장을 생성합니다.
✓
결과: AI가 30일 플랜 생성 Day1~30 구조 + 하루 5문장 규칙으로 즉시 학습 가능한 플랜이 출력됩니다.

사용된 기술

모델 · 백엔드 · 프롬프트 · 데이터

OpenAI GPT-4o-mini

상황/레벨에 맞는 자연스러운 영어 문장 생성에 사용했습니다.

FastAPI 백엔드

빠르고 안정적인 API 서버를 구성하여 프론트엔드와 연결했습니다.

프롬프트 엔지니어링

Day1~30 구조 강제, 하루 5문장 규칙, 영어+한국어 번역 포함 등 출력 포맷을 고정했습니다.

JSON 응답 강제

구조적 데이터로 응답을 받아 UI에서 오류 없이 안정적으로 렌더링되도록 했습니다.

기대효과

학습 경험 개선

맞춤형 학습

사용자의 상황과 수준에 딱 맞는 영어 학습을 제공해 “내가 쓸 표현” 위주로 학습합니다.

실용적 표현

여행/업무/일상 등 실제 필요한 표현을 중심으로 구성해 바로 써먹을 수 있습니다.

초개인화 경험

기존 영어앱이 제공하지 못했던 개인 상황·목표 기반의 새로운 학습 경험을 제공합니다.
